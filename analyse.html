Accessibility

I tested NVDA (NonVisual Desktop Access) on my website to see how it performed. Initially, navigating the site with this method and the keyboard felt unfamiliar, but surprisingly, it went smoothly over time. I could use the tab key to reach all the links on the page. I closed my eyes and managed to navigate to where I wanted. It was a bit challenging when the screen reader spoke Norwegian and my website is in English, but I addressed this in the settings.

I implemented a 'Skip to content' link for screen readers, and I think it works well. I used the example from https://bignerdranch.com/blog/web-accessibility-skip-navigation-links/. When I started adding ARIA to the document, I noticed that NVDA only read the text and not the ARIA. So, I tried Windows' built-in screen reader, but it only read the ARIA and not the text. Finally, I downloaded Chrome Vox as an extension for Chrome. It worked well with both text and ARIA, so I used it to determine where to place ARIA.

Measures taken to increase SEO and UU

  • I removed the button tag from cookies and contact icons.
  • Provided alt text to icons that have a function.
  • Removed alt text from icons and added aria-label since HTML-validator told me to.
  • I fixed the heading level in footer to get better structure.
  • Fixed "login" icon that was categorized as an "empty link". Wrapped it in span, added aria hidden true, and a "go to login site" text that is accessible to screen readers.
  • Changed the alt text on all images to be more detailed.
  • I have not fixed the "reduntant link" issue because it's not a complete functional website.
  • Tested three different screen readers on my site with eyes closed, while using keyboard. Used NVDA, Chrome Vox and Windows screen reader.
  • Added ARIA labels, roles to elements that needed it and kept testing with screen reader
  • Added a "Skip to content" shortcut for screen readers
  • Added microdata to product site and testet it with rich results test. Fixed errors that occured after test.
  • Added meta-description and changed title text.
  • Added meta data for Instagram, Facebook, Twitter and Tiktok. Checked if they worked
